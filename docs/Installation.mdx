# Installation

myui components are distributed via the [shadcn](https://ui.shadcn.com) registry, make sure it's configured before proceeding.

## 1. Configure the `@/` alias

myui components use the `@/` path alias to reference shared utilities and other components after installation. This alias is a [shadcn requirement](https://ui.shadcn.com/docs/installation) and should already be present in your project.

**`tsconfig.json`**

```json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

**`vite.config.ts`**

```ts
import path from 'path'

export default {
  resolve: {
    alias: { '@': path.resolve(__dirname, 'src') },
  },
}
```

## 3. Add the registry

Add the myui registry to your `components.json`:

```json
{
  "registries": {
    "@myui": "https://idos350.github.io/myui/r/{name}.json"
  }
}
```

## 4. Install a component

```bash
pnpm dlx shadcn@latest add @myui/<component-name>
```
